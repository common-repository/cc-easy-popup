== Description ==

[CC Easy Popup] (http://www.codecygnus.com/ccpopup) is a most useful plugin who likes to show up your message or any content you like to show on your specified page via pop over. So that the viewers attraction will be totally in the Popup and they will know the message easily. Its totally easy to use and free popup plugin for the every users. If you need multiple popup to be shown in various pages go for the PRO plugin.

Use the CC Easy popup plugin and created your own beautiful popup. Easily editable in changing the title, content and Background colour for classic and modern version. In trendy version you can add a background image you like. You can also add the social media, Contact form or any shortcode in the text field to popup.

== How to use CC Easy Popup ==

To add the Popup in your website go to the popup settings, Choose your style and add the content to be displayed and save it.

== Installation ==

1. Upload the `CC-Popup` folder to the `/wp-content/plugins/` directory
2. Activate CC Easy Popup plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the `CC Popup` tab that appears in your admin menu.
